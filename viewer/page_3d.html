Components
SENSORY Input
Sensor
Converts external stimulus into spikes
The entry point — nothing happens without input
Relay
Carries signal between regions — long-range wiring
Without relays, clusters are isolated islands
Filter
Sharpens signal locally, suppresses noise
Raw input is messy — filters extract the actual pattern
REACTION Processing
Associator
Forms connections between co-active inputs — learning
The only way to go from reflex to experience
Modulator
Carries reward/punishment, shifts regional sensitivity
Without context, associations are meaningless — "good" vs "bad"
Gate
Hardwired response path — fixed reaction, no learning
Some things can't wait to be learned — predator = flee NOW
Integrator
Combines multiple signal streams into one decision
Real decisions need multiple inputs weighed together
COGNITION Higher
Sustainer
Holds signal active without continuous input — working memory
Can't plan or reason if you forget mid-thought
Predictor
Generates expected signal, outputs the error vs actual
Learning from surprise — the core of cortical computation
Sequencer
Fires in ordered temporal chains — creates sequences
Language, planning, music — anything with an order
Oscillator
Self-firing rhythm — internal clock signal
Coordinates timing across regions, attention, sleep
Suppressor
Actively kills a strong signal — deliberate blocking
Impulse control, focus, "don't do the obvious thing"
MOTOR Output
Planner
Pre-motor — decides WHAT action before sending it
Acting without deciding is just twitching
Commander
Final motor output — drives muscles/behavior
The whole point — thought becomes action
